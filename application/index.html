A clean profile app built for a real deployment workflow.

This application is intentionally simple and polished so the DevOps workflow can shine: build the app, containerize it, push the image to Nexus, and deploy it on a Linux server with Docker Compose.

Node.js Express backend serving API routes and static UI.
MongoDB Profile data persisted in a database container.
Docker Portable image ready for local and cloud deployment.
$ git clone profile-dashboard
$ docker build -t profile-app:1.0 .
$ docker push nexus:8083/profile-app:1.0
$ docker compose up -d
